# PHASE2D25_IMPORT_RESTORE_COPY_ACCESSIBILITY_CONTRACT_V1

Fase 2D.25 define copy prudente y contrato de accesibilidad para la shell.

Mensajes permitidos requeridos:

- Vista previa;
- No se aplicaran cambios;
- Requiere revision;
- Los documentos emitidos o bloqueados estan protegidos;
- Exporta una copia de seguridad antes de cualquier accion futura.

Mensajes prohibidos:

- Restaurar ahora;
- Importar ahora;
- Sobrescribir;
- Aplicado correctamente;
- Seguro al 100%.

Accesibilidad:

- labels de seccion;
- descripciones aria para estado, severidad y acciones;
- texto de disabled actions;
- estado no dependiente solo de color.

- `src/lib/local-data-safety/import-restore-copy.test.ts`
